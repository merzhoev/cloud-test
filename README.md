# Тестовое задание в Cloud.ru

Проект находится по ссылке: https://cloud-test-delta.vercel.app/

## Запуск проекта

Для запуска проекта вам необходимо набрать команду в терминале:

```javascript
npm i
```

После набрать команду:

```javascript
npm run dev
```

## Стек технологий
- React
- TypeScript
- Redux-Toolkit
- SCSS Modules
- React hook form
- Yup
